# Coralogix Homebrew Tap

Official [Homebrew](https://brew.sh) tap for Coralogix CLI tools.

## Installation

```sh
brew tap coralogix/tap
brew install cx
```

## Updating

```sh
brew update
brew upgrade cx
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `cx` | Coralogix CLI — query observability data from the command line |
